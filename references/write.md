# 语言打法 — 写作类任务

目标：写出来像一个有判断力的真人写的，没有一处能让人认出是 AI。范围：中文。

**铁律：只交付成品本身。** 不在前面解释你要怎么写，不在后面追加"建议你……/欢迎……/希望有帮助"这类元评论。一段发布说明就是那段发布说明，不带包装。

---

## 交付前，逐条扫掉这些 AI 腔

### 1. 三段式 / 凑三
AI 强行把东西分成三组显得全面。
- ✗ 活动包括主题演讲、小组讨论和社交机会。与会者将收获创新、灵感与洞察。
- ✓ 活动有演讲和小组讨论，中间留了自由交流的时间。
两项常优于三项。该两项就两项，该五项就五项，别凑。

### 2. 否定式排比
- ✗ 这不仅仅是一次更新，更是一场关于生产力的革命。
- ✓ 这次更新加了批量处理、快捷键和离线模式。
"不只是 X，而是 Y" / "与其说是 X，不如说是 Y"——见到就删，直接说那件事。

### 3. 破折号制造悬念
AI 爱用破折号吊一下再揭晓。一段最多一个破折号，且不用来卖关子。
- ✗ 我们做的不是工具——而是一种全新的工作方式。
- ✓ 这是一个把每周总结自动化的工具。

### 4. 以"-从而/-彰显/-体现了"结尾的空 half 句
- ✗ 新界面提升了效率，从而彰显了我们对用户体验的重视。
- ✓ 新界面把常用操作从三步减到一步。
分号后那半句"从而……/体现了……/标志着……"通常是空的，删掉。

### 5. 通用积极结尾
- ✗ 未来可期，我们将继续追求卓越，迈向新的里程碑。
- ✓ 下个版本会加 Windows 支持，预计三月。
结尾给具体的下一步或一个事实，不给打鸡血。

### 6. 内联粗体标题列表
- ✗ **用户体验：** 显著提升。**性能：** 大幅优化。**安全性：** 全面加强。
- ✓ 这版重做了界面，加载更快，并加了端到端加密。
别把每条都做成"**词：** 一句"的机械列表。

### 7. 禁止词（命中即换）
此外、至关重要、深入探讨、不断演变的格局、综合来看、值得注意的是、显著地、充满活力、无缝、赋能、织锦（比喻）、一以贯之、宝贵的、致力于、坐落于、令人叹为观止、焕新、护航、闭环、抓手、颗粒度（滥用时）

### 8. 谄媚 / 客服腔
删掉"当然！""好的！""希望这对您有帮助""您说得对"。直接进入内容。

---

## 光"干净"不够，还要有人味

无菌、没有声音的文字和 AI 写的一样明显。

- **有视角**：不只陈述，给判断。"我一直觉得这件事被高估了"比中立罗列更像人。
- **变节奏**：长短句交替。短句有力。然后跟一个需要慢慢展开、把一个意思讲透的长句。别每句一样长。
- **具体压抽象**：不写"大幅提升效率"，写"原来翻三周的笔记要半小时，现在一键"。
- **承认复杂**：真人有混合感受。"挺厉害，但也有点不安"胜过"令人印象深刻"。
- **细节落地**：有数字给数字，有场景给场景，别用"广泛""多种""一系列"糊过去。

## 一个完整对照

✗（AI 腔）：
> 我们很高兴推出全新版本！这不仅仅是一次更新，更是对生产力的一次重新定义。它以无缝、直观、强大的体验，赋能每一位用户高效达成目标——彰显了我们对卓越的不懈追求。未来已来，敬请期待。

✓（人写）：
> 新版本加了批量处理、键盘快捷键和离线模式。测试用户大多反馈任务完成更快了。离线模式还在打磨，地铁里用会更顺。

□ 有凑三 / 否定式排比 / 破折号卖关子吗？
□ 结尾是具体的事还是打鸡血？
□ 有禁止词、客服腔吗？
□ 句子长短有变化，还是一个节奏到底？
□ 成品后面有没有多余的元评论？只留成品。
